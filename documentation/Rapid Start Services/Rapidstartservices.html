    Rapid Start Services

    Rapid Start Services is one of the elements in enwis which is used in importing massive amount of data. For example, transferring the ledger entries from the old system by importing this data in the relevant journals and posting them in the new system would be an option and for importing the data into the new system a user can make use of these Rapid Start Packages. The user initially creates a rapid start package by validating the necessary fields and then extracts the information from the old system in the form of an excel sheet and then adds this sheet to the respective package. Then the user can apply the package in the new system to import all the data into the relevant journals and furtherly can post it which helps in reducing a lot of manual work in day-to-day life especially while transferring the data.
